University Prospects is a web application that seeks to help future applicants to Malawian public universities make informed decisions about which programme to apply to by telling users programmes their grades best qualify them for. You can view the site [here](https://www.universityprospects.co/).

### To locally view the site
Clone the repository and using the terminal, open the project's directory and run the following commands

```bash
 cd frontend
 npm install
 npm start 
```
